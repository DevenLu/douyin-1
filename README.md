# 抖音视频抓取
    DoyYinmP4down 是下载入口
    MyCare 是获取个人关注信息，care_url (每个人关注)是动态变化的，需要使用FIDDLER获取
    DouYinConStant 保存了基本的常量
  
## 博客相关描述 ##
	此代码先后修正2次。刚开始是发现下载个人列表时，只是下载前20个，超过20个的就不在下载。
	第二次修正是发现下载的视频不完整。推测是发生了从定向导致。写博客做了简单描述。
